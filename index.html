Why not CLI only?

Non-developers need a safer, app-native path.

1-click deep link

Hermes can use a hermes:// deep link prototype. Codex currently needs its plugin CLI: add this repo as a marketplace, then install the sample plugin.

Permission review

The agent client shows exactly what will be written and what tools or credentials the skill asks for.

Portable fallback

A signed .skill.zip bundle supports drag-and-drop import when client deep links are unavailable.
