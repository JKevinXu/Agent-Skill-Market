# Meeting Notes Summarizer

A sample Agent Skill Market skill that turns raw meeting transcripts into structured summaries, decisions, action items, risks, open questions, and follow-up messages.

## Install

Hermes:

```bash
hermes skills install jkevinxu.meeting-notes-summarizer@1.0.0
```

Codex-compatible clients:

```bash
$skill-installer https://github.com/JKevinXu/Agent-Skill-Market/tree/main/skills/meeting-notes-summarizer
```

Portable bundle:

```bash
curl -L -o meeting-notes-summarizer.skill.zip https://github.com/JKevinXu/Agent-Skill-Market/raw/main/dist/meeting-notes-summarizer.skill.zip
```

## Permissions

- Shell/scripts: none
- Credentials: none
- Network after install: none
- File writes after install: none
